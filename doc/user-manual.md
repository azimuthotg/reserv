## 1. ระบบนี้คืออะไร

ระบบจองพื้นที่บริการ Smart Creative Learning Space คือระบบจองห้องออนไลน์ผ่าน **LINE** สำหรับนักศึกษาและบุคลากรมหาวิทยาลัยนครพนม เพื่อเข้าใช้งานพื้นที่บริการพิเศษชั้น 3 สำนักวิทยบริการ

### ห้องที่ให้บริการ

| ห้อง | ที่ตั้ง | เวลาเปิด–ปิด |
|---|---|---|
| Mini Theater | ชั้น 3 สำนักวิทยบริการ | 08:30–16:30 น. |
| Edutainment | ชั้น 3 สำนักวิทยบริการ | 08:30–16:30 น. |
| Canva Studio | ชั้น 3 สำนักวิทยบริการ | 08:30–16:30 น. |
| Chat GPT Room | ชั้น 3 สำนักวิทยบริการ | 08:30–16:30 น. |
| Meeting Room F1 | ชั้น 3 สำนักวิทยบริการ | 08:30–16:30 น. |

### ผู้ที่ใช้งานได้

- **นักศึกษา** มหาวิทยาลัยนครพนม (มีรหัสนักศึกษาและรหัสผ่านอินเทอร์เน็ต)
- **บุคลากร** มหาวิทยาลัยนครพนม (มีรหัสบุคลากรและรหัสผ่านอินเทอร์เน็ต)

### ความต้องการเบื้องต้น

- มีบัญชี LINE
- โทรศัพท์มือถือที่ติดตั้ง LINE แล้ว หรือ Browser บนคอมพิวเตอร์
- รหัสผู้ใช้ (LDAP) และรหัสผ่านระบบอินเทอร์เน็ตมหาวิทยาลัย

## 2. เริ่มต้นใช้งานครั้งแรก — ลงทะเบียนผูกบัญชี

> **ทำเพียงครั้งเดียว** หลังจากนี้ไม่ต้องลงทะเบียนซ้ำอีก

ก่อนใช้งานระบบจอง ต้องผูกบัญชี LINE เข้ากับรหัสผู้ใช้มหาวิทยาลัยก่อน 1 ครั้ง

### ขั้นตอน

1. เปิด LINE แล้วเข้า **LINE OA ห้องสมุด มนพ.** (Library@NPU)
2. กดปุ่ม **"ระบบจอง"** ในเมนูด้านล่าง หรือเปิดลิงก์: `https://lib.npu.ac.th/reserv/`
3. ระบบจะนำไปยังหน้าลงทะเบียน

![หน้าลงทะเบียน](screenshots/01_register.png)

4. เลือก **ประเภทผู้ใช้งาน**
   - เลือก **"นักศึกษา"** หากเป็นนักศึกษา
   - เลือก **"บุคลากร"** หากเป็นบุคลากรมหาวิทยาลัย
5. กรอก **รหัสผู้ใช้ (LDAP)** เช่น `64010001` (นักศึกษา) หรือรหัสบุคลากร
6. กรอก **รหัสผ่าน** ซึ่งเป็นรหัสผ่านเดียวกับการใช้อินเทอร์เน็ตมหาวิทยาลัย
7. กดปุ่ม **"ลงทะเบียน"**

เมื่อลงทะเบียนสำเร็จ ระบบจะพาไปยังหน้าหลักโดยอัตโนมัติ

> **หมายเหตุ:** หากลืมรหัสผ่านอินเทอร์เน็ต ติดต่อสำนักวิทยบริการหรืองาน IT มหาวิทยาลัย

## 3. หน้าหลัก — ดูห้องและการจองของฉัน

หลังล็อกอินด้วย LINE แล้ว ระบบจะแสดงหน้าหลัก ซึ่งประกอบด้วย 3 ส่วน

![หน้าหลัก](screenshots/02_landing.png)

### 3.1 แถบชื่อผู้ใช้

ด้านบนขวาจะแสดง **ชื่อและคณะ/หน่วยงาน** ของคุณ แสดงว่าล็อกอินสำเร็จแล้ว

### 3.2 การจองของฉัน

แสดงรายการการจองที่**กำลังจะมาถึง** ของคุณ (ตั้งแต่วันนี้เป็นต้นไป) ประกอบด้วย
- ชื่อห้องและเวลาจอง
- ปุ่ม **"ยกเลิกการจอง"** (สำหรับยกเลิกก่อนเวลาเริ่ม)
- การ์ดที่ถูก**ยกเลิกแล้ว**จะแสดงเป็นสีเทาพร้อมป้าย "ยกเลิกแล้ว"

> การ์ดการจองจะหายไปเองหลังผ่านวันที่จองไปแล้ว

### 3.3 รายการห้อง

แสดงห้องทั้งหมดที่เปิดให้บริการ แต่ละห้องมี
- ชื่อห้องและที่ตั้ง
- เวลาเปิด–ปิดบริการ
- ปุ่ม **"จองเลย"** — คลิกเพื่อจองห้องนั้น
- ลิงก์ **"ตารางการใช้ห้อง"** — ดูปฏิทินของห้องนั้นโดยเฉพาะ

## 4. จองห้อง

### ขั้นตอน

1. จากหน้าหลัก กดปุ่ม **"จองเลย"** ที่ห้องที่ต้องการ

![หน้าจองห้อง](screenshots/03_booking.png)

2. ระบบจะแสดงหน้าจองห้อง กรอกข้อมูลดังนี้

   | ช่อง | รายละเอียด |
   |---|---|
   | **วันที่จอง** | เลือกวันที่ต้องการ (จองล่วงหน้าได้สูงสุด 5 วัน เฉพาะวันจันทร์–ศุกร์) |
   | **เวลาเริ่ม** | เวลาเริ่มใช้งาน เช่น 09:00 |
   | **เวลาสิ้นสุด** | เวลาเลิกใช้งาน เช่น 12:00 |
   | **ชื่อกลุ่ม / วัตถุประสงค์** | ระบุชื่อกลุ่มหรือกิจกรรมที่จะทำ เช่น "ชมรมถ่ายภาพ" |
   | **จำนวนผู้เข้าใช้** | จำนวนคนที่จะเข้าใช้ห้อง |

3. กดปุ่ม **"ยืนยันการจอง"**

![จองสำเร็จ](screenshots/04_success.png)

4. หากสำเร็จ จะแสดงหน้า **"จองสำเร็จแล้ว!"** พร้อมสรุปข้อมูลการจอง
5. ระบบจะส่ง **แจ้งเตือน LINE** ยืนยันการจองให้ทันที

### ข้อกำหนดการจอง

- จองล่วงหน้าได้ **สูงสุด 5 วันทำการ**
- จองได้เฉพาะ **วันจันทร์–ศุกร์** เท่านั้น (ไม่รวมวันหยุดราชการ)
- เวลาจองต้องอยู่ในช่วง **08:30–16:30 น.**
- ไม่สามารถจองซ้อนกับการจองอื่นที่มีอยู่แล้วได้

## 5. ยกเลิกการจอง

หากต้องการยกเลิกการจองก่อนเวลาเริ่ม สามารถทำได้จากหน้าหลัก

### ขั้นตอน

1. จากหน้าหลัก มองหาการ์ดการจองในส่วน **"การจองของฉัน"**
2. กดข้อความ **"ยกเลิกการจอง"** (สีแดง) ใต้ชื่อการจอง

![ยกเลิกการจอง](screenshots/05_cancel.png)

3. ระบบจะแสดงหน้าต่างยืนยัน พร้อมข้อมูลการจองที่จะยกเลิก
4. กดปุ่ม **"ยกเลิกการจอง"** เพื่อยืนยัน หรือ **"ย้อนกลับ"** หากเปลี่ยนใจ
5. เมื่อยกเลิกสำเร็จ การ์ดจะเปลี่ยนเป็นสีเทาพร้อมป้าย **"ยกเลิกแล้ว"**
6. ระบบจะส่ง **แจ้งเตือน LINE** ยืนยันการยกเลิกให้ทันที

### ข้อควรระวัง

> **ไม่สามารถยกเลิกได้** เมื่อถึงเวลาเริ่มใช้งานแล้ว หรือหลังจากวันที่จองผ่านไปแล้ว

## 6. ปฏิทินรวม — ดูตารางการใช้ห้อง

ปฏิทินรวมแสดงการจองทุกห้องในมุมมองรายเดือน ใช้วางแผนการจองได้ง่ายขึ้น

### วิธีเข้าถึง

- กดแบนเนอร์ **"ดูตารางการจองทั้งหมด"** บนหน้าหลัก หรือ
- เปิดตรงที่ URL: `https://lib.npu.ac.th/reserv/calendar/`

![ปฏิทินรวม](screenshots/06_calendar.png)

### การใช้งาน

- **ดูทุกห้อง:** กดปุ่ม "ทุกห้อง" (สีเขียว)
- **กรองตามห้อง:** กดปุ่มชื่อห้องที่ต้องการดู เช่น "Mini Theater"
- **เปลี่ยนเดือน:** กดปุ่ม **‹** (เดือนก่อน) หรือ **›** (เดือนถัดไป)
- **สีของแถบ:** แต่ละห้องมีสีเฉพาะ ดูได้จาก Legend ด้านล่างปฏิทิน

## 7. บัตรสมาชิกดิจิทัล (Virtual Card)

บัตรสมาชิกดิจิทัลแสดงข้อมูลประจำตัวของคุณพร้อม QR Code สำหรับเข้าใช้บริการห้องสมุด

### วิธีเปิดบัตร

1. เปิด LINE OA ห้องสมุด มนพ.
2. กดปุ่ม **"โปรไฟล์"** ในเมนูด้านล่าง

![บัตรสมาชิกดิจิทัล](screenshots/07_card.png)

### ข้อมูลที่แสดงบนบัตร

| รายการ | คำอธิบาย |
|---|---|
| รูปโปรไฟล์ | รูปจากบัญชี LINE ของคุณ |
| ชื่อ–นามสกุล | ชื่อจริงจากระบบมหาวิทยาลัย |
| ประเภท | นักศึกษา หรือ บุคลากร |
| คณะ / หน่วยงาน | คณะ (สำหรับนักศึกษา) หรือหน่วยงาน (สำหรับบุคลากร) |
| สถานะสมาชิก | ✓ สมาชิกห้องสมุด (สีเขียว) หรือ ✗ ไม่ใช่สมาชิก (สีแดง) |
| QR Code | รหัสสำหรับสแกนเข้าประตูห้องสมุด |

### การใช้ QR Code

นำ QR Code ให้เจ้าหน้าที่หรือเครื่องสแกนที่หน้าประตูสแกน เพื่อยืนยันตัวตนเข้าใช้บริการห้องสมุด

## 8. การแจ้งเตือนผ่าน LINE

ระบบจะส่งข้อความแจ้งเตือนผ่าน LINE OA ในสถานการณ์ต่าง ๆ โดยอัตโนมัติ

![การแจ้งเตือน LINE](screenshots/08_notification.png)

### ประเภทการแจ้งเตือน

| เวลา | ข้อความ |
|---|---|
| ทันทีหลังจอง | ✅ ยืนยันการจองสำเร็จ พร้อมรายละเอียดห้อง วันที่ เวลา |
| ก่อนเริ่ม 15 นาที | ⏰ เตือนให้เดินทางมาถึงทัน |
| ก่อนหมดเวลา 10 นาที | ⏳ เตือนให้เก็บสิ่งของออกจากห้อง |
| เมื่อถูกยกเลิกโดยเจ้าหน้าที่ | ❌ แจ้งว่าการจองถูกยกเลิก พร้อมเหตุผล |
| เมื่อยกเลิกด้วยตนเอง | ❌ ยืนยันการยกเลิกที่คุณกดเอง |

> **หมายเหตุ:** ตรวจสอบให้แน่ใจว่าไม่ได้ Mute การแจ้งเตือนจาก LINE OA Library@NPU

## 9. คำถามที่พบบ่อย (FAQ)

**Q: ลืมรหัสผ่านอินเทอร์เน็ตมหาวิทยาลัย ทำอย่างไร?**
A: ติดต่องาน IT มหาวิทยาลัยนครพนม หรือสำนักวิทยบริการโดยตรง

**Q: จองแล้วแต่ไม่ได้รับแจ้งเตือน LINE ทำอย่างไร?**
A: ตรวจสอบว่าได้ Follow LINE OA Library@NPU แล้ว และไม่ได้ Mute การแจ้งเตือน

**Q: จองห้องวันเสาร์–อาทิตย์ได้ไหม?**
A: ไม่ได้ครับ ระบบให้จองเฉพาะวันจันทร์–ศุกร์เท่านั้น

**Q: ลงทะเบียนแล้ว แต่พอเปิดใหม่ต้องล็อกอินอีกไหม?**
A: ถ้าเปิดผ่าน LINE ไม่ต้องล็อกอินซ้ำ ระบบจะจำบัญชีให้อัตโนมัติ ถ้าเปิดผ่าน Browser อาจต้องล็อกอิน LINE ครั้งแรก

**Q: จองซ้อนกับคนอื่นได้ไหม?**
A: ไม่ได้ครับ ระบบจะตรวจสอบการจองซ้อนทับโดยอัตโนมัติ หากมีคนจองช่วงเวลานั้นแล้ว จะแจ้งว่า "เวลานี้มีผู้จองแล้ว"

**Q: อยากเปลี่ยนเวลาจอง ทำได้ไหม?**
A: ไม่มีฟังก์ชันแก้ไขโดยตรง ให้ยกเลิกการจองเดิมแล้วจองใหม่ในเวลาที่ต้องการ

**Q: สมาชิก Walai ขึ้น "ไม่ใช่สมาชิก" แต่เป็นสมาชิกจริง?**
A: ติดต่อเจ้าหน้าที่ห้องสมุดเพื่อตรวจสอบข้อมูลในระบบ Walai

**Q: บัตรสมาชิกดิจิทัล (Virtual Card) ใช้แทนบัตรนักศึกษาได้ไหม?**
A: ใช้ได้สำหรับการเข้าใช้บริการห้องสมุดเท่านั้น ไม่สามารถใช้แทนบัตรนักศึกษาในกรณีอื่น
