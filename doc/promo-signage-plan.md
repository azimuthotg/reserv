## 2. House Style Guide (โทนรวมทุกป้าย)

อิงจาก `example1.jpg` เพื่อให้ทุกป้ายเป็นชุดเดียวกัน

### สี
| บทบาท | สี (HEX) | ใช้ตรงไหน |
|---|---|---|
| น้ำเงินหลัก (เข้ม) | `#1A3A8F` | พื้นหลัง gradient ด้านบน |
| น้ำเงินไล่ (อ่อน) | `#2E6BD6` | พื้นหลัง gradient ด้านล่าง |
| ทอง/ส้ม accent | `#F5A623` | กรอบหัวเรื่อง, ปุ่ม, ไฮไลต์เลข |
| เหลืองสว่าง | `#FFD24C` | ขอบ badge, ดาว/sparkle |
| ขาว | `#FFFFFF` | ตัวอักษรหลัก, การ์ด badge |
| เขียว LINE | `#06C755` | เฉพาะคำว่า/แผง "LINE OA" |

### ฟอนต์ (ไทย)
- หัวเรื่อง: **Kanit / Prompt Bold** (หนา โค้งมน)
- เนื้อความ: **Sarabun / Prompt Regular**
- หลีกเลี่ยงฟอนต์อังกฤษล้วน เพราะ AI gen มักทำไทยเพี้ยน → ถ้า gen ด้วย AI ให้พิมพ์ข้อความไทยทับเองในขั้นแก้

### องค์ประกอบประจำ (ใส่ทุกป้าย)
1. **มุมบนซ้าย:** โลโก้ ARC•NPU (ทอง) + ข้อความ 2 บรรทัด "สำนักวิทยบริการ / มหาวิทยาลัยนครพนม"
2. **มาสคอต:** ตัวการ์ตูนน่ารัก นักศึกษา/บรรณารักษ์ ใส่เสื้อโปโลน้ำเงิน ถือมือถือโชว์แอปจอง
3. **พื้นหลัง:** ไล่สีน้ำเงิน + sparkle ขาว + จุดทอง + เส้น line-art อาคารมหาวิทยาลัยจาง ๆ ที่มุม
4. **Badge:** การ์ดขาวขอบมน เงานุ่ม เลขในวงกลมทอง
5. **แถบล่าง (contact bar):** พื้นทอง/ขาว ไอคอน + ข้อมูลติดต่อ (ดูข้อ 4)
