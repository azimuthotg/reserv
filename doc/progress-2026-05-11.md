### Poster Version 2 — อัพเดทเนื้อหา

ปรับปรุง `doc/poster-content.md` และ `doc/poster-booking-guide.html` ให้รองรับ **2 ช่องทางการจอง** (เดิมรองรับเฉพาะ LINE OA)

| ส่วนที่แก้ไข | การเปลี่ยนแปลง |
|---|---|
| Header / Pill badge | เพิ่ม pill "เว็บไซต์" คู่กับ "LINE OA" |
| Channel strip | เพิ่มแถบ 2 ช่องทาง (LINE OA / เว็บไซต์) ใหม่ทั้งหมด |
| Step 1 | เปลี่ยนเป็น "เลือกช่องทางการจอง" อธิบาย 2 เส้นทาง |
| Step 2 | ระบุว่าผูกบัญชีครั้งเดียวใช้ได้ทั้ง 2 ช่องทาง |
| Step 3 | ระบุว่าจองได้ทั้ง LINE OA และเว็บไซต์ |
| Step 4 | เปลี่ยนชื่อ "รับการยืนยันทาง LINE" → "รับการยืนยันการจอง" |
| Step 5 | Check-in ได้ทั้ง LINE และเว็บไซต์ |
| Note IoT | ควบคุมอุปกรณ์ได้ทั้ง LINE หรือเว็บไซต์ |

### Poster PNG — สร้างด้วย canvas-design skill

ติดตั้ง skill `canvas-design` ผ่าน `skillfish` และสร้างโปสเตอร์ PNG โดยใช้ Python + Pillow

**ไฟล์ที่สร้าง:**
- `doc/poster-v2-philosophy.md` — Visual Philosophy "Neon Meridian"
- `doc/poster-v2.png` — A4 (1240×1754px @ 150dpi) dark mode, LINE green

**เทคนิคที่ใช้:**
- Font ภาษาไทย: Leelawadee (`/mnt/c/Windows/Fonts/leelawad.ttf`)
- Font อังกฤษ: Outfit, GeistMono, BigShoulders จาก canvas-fonts
- Icon ทุกตัววาดด้วย Pillow geometry (แทน emoji ที่ไม่ render บน Linux)
- Dot grid, radial glow, gradient connector lines ทำด้วย RGBA overlay
