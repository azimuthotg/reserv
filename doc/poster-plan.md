# แผนชุดป้ายประชาสัมพันธ์ระบบจองพื้นที่บริการ

ระบบจองพื้นที่บริการ Smart Creative Learning Space

สำนักวิทยบริการ มหาวิทยาลัยนครพนม

## เป้าหมาย

ทำให้กลุ่มเป้าหมายทราบว่า:

- สำนักวิทยบริการมีระบบจองพื้นที่บริการ
- นักศึกษา อาจารย์ และบุคลากรสามารถใช้งานได้
- จองได้ 2 ช่องทาง: LINE OA และเว็บไซต์
- ผู้ใช้สามารถสแกน QR Code และเริ่มจองได้ทันที

## ชุดป้ายที่วางแผนไว้

ออกแบบต้นฉบับทั้งหมด `8` แบบ:

| ลำดับ | แบบป้าย | จำนวนต้นฉบับ | จุดประสงค์ |
|---|---|---:|---|
| 1 | ป้ายเปิดตัวระบบ | 1 | ทำให้ทราบว่ามีบริการและเริ่มจองได้ทันที |
| 2 | ป้ายวิธีจอง | 1 | อธิบายขั้นตอนใช้งาน 3–5 ขั้น |
| 3 | ป้ายกติกาสรุป | 1 | สรุปเวลาเปิดบริการ ระยะเวลาจอง และเงื่อนไขสำคัญ |
| 4 | ป้ายหน้าห้อง Mini Theater | 1 | แนะนำห้องและสแกนจองห้องนี้โดยตรง |
| 5 | ป้ายหน้าห้อง Edutainment Zone | 1 | แนะนำห้องและสแกนจองห้องนี้โดยตรง |
| 6 | ป้ายหน้าห้อง Canva Pro Studio | 1 | แนะนำห้องและสแกนจองห้องนี้โดยตรง |
| 7 | ป้ายหน้าห้อง ChatGPT Station | 1 | แนะนำห้องและสแกนจองห้องนี้โดยตรง |
| 8 | ป้ายหน้าห้องประชุมชั้น 1 | 1 | แนะนำห้องและสแกนจองห้องนี้โดยตรง |

ป้ายหลัก 3 แบบแรกสามารถปรับสัดส่วนต่อยอดสำหรับ Facebook, LINE OA และจอประชาสัมพันธ์ได้

## แนวทางภาพรวม

- ใช้บริบทจริงของสำนักวิทยบริการ เพื่อให้ผู้พบเห็นเข้าใจทันทีว่าเป็นบริการของห้องสมุด
- ใช้ QR Code จริงที่ทดสอบแล้ว ไม่สร้าง QR Code จำลองในภาพ
- ใช้ภาพบรรณารักษ์จริงเป็น reference แล้วแปลงเป็นตัวการ์ตูนน่ารัก
- ใช้ภาพพื้นที่จริงเป็น reference สำหรับฉากหลัง เช่น ทางเข้า เคาน์เตอร์บริการ หรือชั้นหนังสือ
- รักษาพื้นที่ข้อความให้โล่ง อ่านได้เร็วจากระยะห่าง
- แยกภาพประกอบออกจากข้อความและ QR Code เพื่อให้แก้ไขข้อความได้โดยไม่ต้องสร้างภาพใหม่ทั้งหมด

## ป้ายที่ 1 — ป้ายเปิดตัวระบบ

### หน้าที่ของป้าย

ทำให้ผู้พบเห็นทราบภายในไม่กี่วินาทีว่า:

1. สำนักวิทยบริการมีระบบจองพื้นที่ Smart Creative Learning Space
2. นักศึกษา อาจารย์ และบุคลากรสามารถใช้งานได้
3. เริ่มจองผ่าน LINE OA หรือเว็บไซต์ได้
4. สแกน QR Code เพื่อเริ่มใช้งานได้ทันที

ป้ายนี้ยังไม่ควรใส่วิธีจองแบบละเอียด เพราะจะมีป้ายวิธีจองแยกต่างหาก

### โครงข้อความเบื้องต้น

**หัวเรื่อง:**

จองพื้นที่ Smart Creative Learning Space ได้แล้ววันนี้

**ข้อความรอง:**

สำหรับนักศึกษา อาจารย์ และบุคลากร มหาวิทยาลัยนครพนม

**ข้อความช่องทาง:**

จองง่าย ๆ ผ่าน LINE OA หรือเว็บไซต์

**คำกระตุ้น:**

สแกนเพื่อเริ่มจอง

### วัตถุดิบที่ต้องเตรียม

#### จำเป็น

1. โลโก้มหาวิทยาลัยนครพนม ไฟล์คมชัด พื้นหลังโปร่งใส (`PNG` หรือ `SVG`)
2. โลโก้สำนักวิทยบริการ หากมี ไฟล์คมชัด พื้นหลังโปร่งใส
3. QR Code จริงสำหรับ LINE OA ไฟล์ `PNG` ความละเอียดสูง
4. ลิงก์ปลายทางจริงของ QR Code LINE OA สำหรับทดสอบก่อนใช้
5. QR Code จริงสำหรับเว็บไซต์ `https://lib.npu.ac.th/reserv/` ไฟล์ `PNG` ความละเอียดสูง
6. ภาพถ่ายบรรณารักษ์จริงที่ยินยอมให้ใช้เป็น reference อย่างน้อย 2–3 ภาพ
7. ภาพถ่ายบริบทสำนักวิทยบริการจริงอย่างน้อย 3–5 ภาพ

#### ภาพบรรณารักษ์ที่เหมาะสำหรับทำการ์ตูน

- ภาพหน้าตรงหรือหันเล็กน้อย เห็นใบหน้าชัด
- ภาพครึ่งตัวหรือเต็มตัว
- เห็นชุดทำงานจริง ป้ายชื่อ หรือรายละเอียดที่ช่วยให้จดจำได้
- แสงสว่างพอ ไม่ย้อนแสง
- หากต้องการตัวละครมากกว่า 1 คน ให้ส่งภาพแต่ละคนแยกกัน

#### ภาพสถานที่ที่เหมาะสำหรับเป็นฉาก

- ทางเข้าอาคารหรือป้ายชื่อสำนักวิทยบริการ
- เคาน์เตอร์บริการที่ผู้ใช้งานเห็นจริง
- มุมชั้นหนังสือหรือพื้นที่นั่งอ่าน
- มุมที่มีเอกลักษณ์ของพื้นที่ Smart Creative Learning Space
- ภาพแนวนอนและแนวตั้งอย่างละอย่างน้อย 1 ภาพ

#### มีแล้วจะช่วยให้ออกแบบแม่นขึ้น

- คู่มือสีหรือสีประจำหน่วยงาน
- ตัวอย่างป้ายเดิมที่ต้องการให้กลมกลืนกับพื้นที่
- ขนาดงานพิมพ์หลัก เช่น `A3` แนวตั้ง หรือขนาดตั้งโต๊ะ
- จุดติดตั้งจริง พร้อมภาพบริเวณที่จะติดตั้ง
- ไฟล์ QR Code ที่ต้องการใช้เป็น QR หลักและ QR รอง
- ความต้องการให้ตัวละครบรรณารักษ์ถือโทรศัพท์ หนังสือ หรือยืนหน้าเคาน์เตอร์

### แนวทางจัดวางเบื้องต้น

- ส่วนบน: โลโก้และชื่อสำนักวิทยบริการ
- ส่วนกลาง: ตัวการ์ตูนบรรณารักษ์จาก reference จริง พร้อมฉากห้องสมุดจริงแบบการ์ตูน
- ข้อความเด่น: “จองพื้นที่ Smart Creative Learning Space ได้แล้ววันนี้”
- ส่วนล่าง: QR Code จริง พร้อมป้ายกำกับ LINE OA และเว็บไซต์
- ใช้ LINE OA เป็นช่องทางหลัก และเว็บไซต์เป็นช่องทางรอง

## ข้อมูลสาธารณะที่ตรวจสอบแล้ว

- เว็บไซต์สำนักวิทยบริการ: `https://arc.npu.ac.th/`
- เว็บไซต์แสดงบริการออนไลน์ “ระบบจองห้อง”
- เวลาเปิดบริการที่เว็บไซต์เผยแพร่:
  - จันทร์–ศุกร์ `08:30–16:30 น.`
  - เสาร์–อาทิตย์ `09:00–17:00 น.`
- ที่อยู่ที่เว็บไซต์เผยแพร่:
  - `167 หมู่ที่ 8 บ้านเนินสะอาด ตำบลนาราชควาย อำเภอเมือง จังหวัดนครพนม 48000`

## ลำดับการทำงานต่อ

1. เตรียมวัตถุดิบจริงสำหรับป้ายเปิดตัวระบบ
2. ตรวจ QR Code จริงทุกปลายทาง
3. สรุปข้อความสุดท้ายและเลือกแนวภาพ
4. สร้างภาพตัวการ์ตูนจาก reference จริง โดยไม่ใส่ตัวอักษรหรือ QR Code ลงในภาพที่ generate
5. ประกอบป้ายให้แก้ไขข้อความและ QR Code ได้
6. ตรวจความอ่านง่ายและสแกน QR Code จากงานขนาดจริง
