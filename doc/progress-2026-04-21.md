## สิ่งที่ดำเนินการในวันนี้

### 1. เพิ่มฟิลด์รายละเอียดห้อง (Room Detail Fields)

เพิ่ม field ใหม่ใน Model `Room` จำนวน 6 field เพื่อรองรับการแสดงข้อมูลห้องให้ครบถ้วน อ้างอิงตามหน้า [arc.npu.ac.th/services/mini-theater-room](https://arc.npu.ac.th/services/mini-theater-room)

| Field | ประเภท | คำอธิบาย |
|---|---|---|
| `min_attendees` | IntegerField | จำนวนผู้ใช้ขั้นต่ำ |
| `max_booking_hours` | IntegerField | เวลาจองสูงสุดต่อครั้ง (ชั่วโมง) |
| `eligible_users` | TextField | ผู้มีสิทธิ์ใช้บริการ |
| `how_to_use` | TextField | ขั้นตอนการใช้บริการ |
| `facilities` | TextField | อุปกรณ์ / สิ่งอำนวยความสะดวก |
| `rules` | TextField | กฎระเบียบการใช้ห้อง |

**Migration ที่สร้าง:**
- `0005_room_add_detail_fields` — fields ชุดแรก 5 field
- `0006_room_add_how_to_use` — เพิ่ม `how_to_use` ภายหลัง

### 2. ปรับหน้าฟอร์มจัดการห้อง (Room Form)

**ไฟล์:** `booking/forms.py`, `booking/templates/booking/manage/room_form.html`

ออกแบบฟอร์มใหม่จาก generic `{% for field in form %}` เป็น layout แบบ section-based 4 ส่วน:

| Section | Fields |
|---|---|
| **ข้อมูลทั่วไป** | ชื่อห้อง, Booking Key, คำอธิบาย, ที่ตั้ง |
| **กำหนดการใช้งาน** | เวลาเปิด/ปิด, เวลาสูงสุด, ความจุ, ขั้นต่ำ, ผู้มีสิทธิ์ |
| **รายละเอียดบริการ** | ขั้นตอนการใช้บริการ, อุปกรณ์, กฎระเบียบ |
| **การตั้งค่า** | HA Entity ID, สถานะเปิด/ปิด |

### 3. หน้ารายละเอียดห้อง (Room Detail Page)

**URL:** `/reserv/room/<booking_name>/`
**ไฟล์ใหม่:** `booking/templates/booking/room_detail.html`

- Public page ไม่ต้อง login
- แสดงรูปห้อง (full-width hero) + accent bar สีประจำห้อง
- Chip ข้อมูลสรุป: เวลาเปิด/ปิด, ความจุ, ขั้นต่ำ, ชั่วโมงสูงสุด
- Section ผู้มีสิทธิ์ใช้บริการ
- Section ขั้นตอนการใช้บริการ (numbered list พร้อมสีประจำห้อง)
- Section อุปกรณ์ / สิ่งอำนวยความสะดวก (bullet list)
- Section กฎระเบียบ (bullet list สีแดง)
- Sticky footer ปุ่ม "จองห้องนี้"

---

### 4. เพิ่มปุ่ม "รายละเอียด" ในการ์ดหน้าจอง

**ไฟล์:** `booking/templates/booking/landing.html`

การ์ดแต่ละห้องในหน้าแรกเพิ่มปุ่ม **"รายละเอียด"** (outline style) คู่กับปุ่ม "จองห้องนี้" เพื่อให้ผู้ใช้สามารถดูข้อมูลห้องก่อนตัดสินใจจอง

```
[รายละเอียด]  [จองห้องนี้]
```
